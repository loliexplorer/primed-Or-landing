## Story
**As a** Branding Manager,
**I want** a high-performance landing page with a cinematic hero section,
**so that** visitors are immediately impressed by the brand's premium aesthetic and "Soul of Nature" philosophy.

## Acceptance Criteria
1.  [ ] **Project Setup:** Next.js (App Router) project initialized with TypeScript, Tailwind CSS, and ESLint.
2.  [ ] **Fonts & Theme:** "Playfair Display" (Serif) and "Inter" (Sans) configured. Global CSS variables for Gold/Earth colors implemented.
3.  [ ] **Structure:** Component folders created per `ui-architecture.md` (ui, features, layout).
4.  [ ] **Hero Video:** Background video plays smoothly on load (autoplay, loop, muted).

6.  [ ] **Performance:** LCP under 2.5s (Video should use a poster image placeholder).
7.  [ ] **Responsive:** Looks good on Mobile (375px) and Desktop.

## Dev Notes
*   **Source Tree:** Refer to `docs/ui-architecture.md#project-structure` for exact folder layout.
*   **Styling:** Use variables `--primary-gold` and `--background` defined in `ui-architecture.md#styling-guidelines`.
*   **Video:** For the Dev phase, you can use a placeholder video URL (e.g., from Pexels or a solid color with text 'Video Placeholder') if the actual asset isn't ready. **Use `next/image` for the poster.**
*   **Animation:** Use `initial={{ opacity: 0 }}` and `animate={{ opacity: 1 }}` for the simple entrance.
*   **Strict Rule:** No `console.log`.

## Testing
*   **Unit:** Test `HeroSection` renders without crashing.
*   **Visual:** Verify fonts are correct (Serif for headings, Sans for body).
*   **Manual:**
    1.  Open page -> Video starts playing immediately.
    2.  Refresh -> Text fades in smoothly.
    3.  Resize to mobile -> Text is readable, video covers background.
