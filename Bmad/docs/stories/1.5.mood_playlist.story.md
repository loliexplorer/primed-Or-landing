# Story 1.5: Mood Playlist - "Sound of Sanctuary"

## Status
Cancelled (Removed by User Request)

## Story
**As a** User looking for relaxation,
**I want** to listen to curated background sounds (Rain, Forest, Soft Piano),
**so that** I can immerse myself in the "Primed'Or atmosphere" while browsing.

## Acceptance Criteria
1.  [ ] **UI Component:** A fixed/sticky "Floating Player" or a dedicated section on the page.
    *   *Decision:* Let's do a **Floating Widget** (Bottom Right) so it persists across scrolling.
2.  [ ] **Playlist:** 3 Tracks:
    *   *Morning Dew* (Nature sounds + Soft melody).
    *   *Deep Focus* (White noise / Rain).
    *   *Midnight Jazz* (Relaxing Jazz).
3.  [ ] **State Management:** Playing state must persist (using Zustand).
    *   Audio should not stop unpredictably (though navigation in Next.js App Router keeps layout mounted, so persistent layout is easy).
4.  [ ] **Interaction:** Play/Pause toggle, Next Track.
5.  [ ] **Visuals:** A mini music visualizer animation (bars jumping) when playing.
